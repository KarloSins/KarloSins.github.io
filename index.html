Ce projet prend la suite d’un projet similaire, mis à bien par les élèves de terminale de l’année dernière. L’objectif est de pouvoir reconnaitre un badge associé à une personne sur une base de données se trouvant sur internet. Pour ce travail, nous avons fait le choix d’utiliser un Raspberry Py 4, un lecteur NFC (= Near Field Communication) et les différents badges fournis avec. À la différence des élèves de l’année dernière, nous optons pour le NFC, là où ils avaient utilisé le RFID (=Radio Frequency IDentification).
